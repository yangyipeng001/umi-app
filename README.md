# umi project

## Getting Started

Install dependencies,

```bash
# 安装 tyarn
$ npm i yarn tyarn -g
```

```bash
$ yarn 或 tyarn
```

Start the dev server,

```bash
$ yarn start 或 tyarn start
```


```bash
# umi 3.0 接入sass
# 先安装 
$ yarn install @umijs/plugin-sass --save

# 再安装
$ yarn install node-sass sass-loader --save
```
